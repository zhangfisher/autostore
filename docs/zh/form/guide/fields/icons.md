# Number

图标选择

## 参数

```ts
type SchemaIconsWidgetOptions = {
    // 使用,分割多个图标
    icons?: string | string[];
    // 是否包括内置图标
    builtIn?: boolean;
    // 下拉选择
    dropdown?: boolean;
    // 显示大小，默认24px
    size?: string;
    // 是否多选
    multiple?: boolean;
};
```
